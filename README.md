# Currency-Convert

## Proje Açıklaması

Bu projemde Java dilinde API&JSON kullanarak "fixer.io" internet adresinden güncel döviz kurlarını 
alıp, "Euro" baz alınarak diğer kurları gösteren basit bir uygulama yapmaya çalıştım. <br/>
